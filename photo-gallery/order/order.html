For the order page, I decided to use tables for the layout. Most sources recommend not using tables for layout, but I decided to try to see how this would turn out. While I was able to achieve the layout I was looking for, I understand why using tables for layout is discouraged. It's definitely an inefficient method, and the same result can be achieved using other methods. It seems to work well enough when designing forms, and this would probably be the only situation I would consider using tables for layout.

One of the advantage of flexbox and Bootstrap is the ease of responsive design which is lost when using tables for layout.
