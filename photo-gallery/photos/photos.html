For the Gallery section of the website, I've used the "container" class in Bootstrap. This class makes use of the display:flex property which I've found to be an excellent tool for layouts; especially with rows and columns of content. I've often used diplay:flex in the past on it's own, but the some of the features of Bootstrap make it even better. Most of the time if I use Bootstrap I still include my own style sheet to have more control of the CSS. Although I do not have experience making responsive websites, using the container-fluid class in Bootstrap provides minimal responsive functionality.

The flex box approach is perfect when displaying content as seen on the page below. I would find it useful when designing any page that involves products being sold. While it is certainly possible to achieve the same layout with different methods, I believe this is the most efficient with plenty of flexibility. In most situations I think this would be my goto tool for layout considering the ease of use and versatility.
