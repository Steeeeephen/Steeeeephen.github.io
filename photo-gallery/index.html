Photography

The main purpose of this website is to present photography.

For this section of the website, I've made use of the "float" and "clear" properties to achieve the layout I was looking for. I have not made use of the float property in the past, so I wanted to make a point of using it for this assignment. While it is a simple and effective solution for the layout I wanted in this section of the website, I did have some problems with some content I wanted to have after the "banner_main" div in my code. Using the "clear" property allowed me to display the content in the "examples" div below.

The "sample photos" below are all links to different photographs displayed as inline elements. These are fairly simple methods that achieve the design I was looking for on the home page. It became evident that if I wanted to have more content displayed in different ways, it may have become more difficult using the float property and inline layout. I would most likely use these methods for when I am designing a more simple page.
